# 🚀 Advanced Java Practical Builder (Pro Studio)

A professional, high-performance web environment for building and exporting perfect Java Practical Files. Featuring a modern IDE-inspired layout, cloud synchronization, and premium academic themes.

## ✨ Key Features

### 💎 Pro Studio IDE Layout
*   **Symmetrical Sidebars**: Modern left-hand Page Explorer and right-hand Tool Panel.
*   **Glassmorphism UI**: High-end `backdrop-filter` blur effects with sleek dark mode aesthetics.
*   **Invisible Scrollbars**: Immersive, edge-to-edge experience with hidden native scrollbars.

### 📜 Document Management
*   **Dynamic Pagination**: Add, reorder, and move pages with real-time thumbnail updates.
*   **Academic Themes**: 10+ professional themes including *Academic Clean*, *Cornell Notes*, *Yellow Legal Pad*, and *Graph Paper*.
*   **Live Preview**: Instant visual feedback as you type code or descriptions.

### ☁️ Cloud Portal Integration
*   **PHP/MySQL Secure Sync**: Save and load your practical files from any device.
*   **Auto-Save Engine**: Reliable state preservation to prevent data loss.

---

## 🛠️ Tech Stack

*   **Frontend**: HTML5, Vanilla JavaScript (Modern ES6+), Phosphor Icons.
*   **Styling**: Advanced CSS3 (Glow filters, Glassmorpism, Flexbox/Grid).
*   **Backend**: PHP 7.4+ (RESTful API).
*   **Database**: MySQL (PDO safe-queries).

## 🚀 Getting Started

### Prerequisites
*   A web server (Shared hosting like **InfinityFree** or local XAMPP/WAMP).
*   PHP 7.4 or higher.
*   MySQL Database.

### Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/ashmitsingh001/practical-file-builder.git
    ```
2.  **Database Setup**:
    *   Import the `Practical_Portal/database_setup.sql` into your PHPMyAdmin.
3.  **Configuration**:
    *   Open `Practical_Portal/config.php`.
    *   Replace placeholders with your real Database credentials.
4.  **Upload**:
    *   Upload all files to your server's `htdocs` or `public_html` directory.
